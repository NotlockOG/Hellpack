How It Works

📦

Base64 Embed

All WAD, PK3, DEH, and config files are base64-encoded and embedded directly into the HTML as JS data arrays.

💾

Virtual FS

Emscripten's virtual filesystem is pre-populated with all files at runtime. PK3 ZIPs are extracted into a virtual directory tree.

🎮

WASM Engine

Chocolate Doom compiled to WebAssembly handles all rendering and game logic — no plugins, no install.

🌐

Offline Play

Open the HTML in Chrome, Firefox, or Edge. No internet required after the engine is cached by your browser.
